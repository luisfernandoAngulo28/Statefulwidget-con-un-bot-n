# Crear un Statefulwidget con un botón que cambie ya sea color o tamaño en pantalla (20 puntos)

El estudiante deberá desarrollar una aplicación básica en Flutter utilizando un StatefulWidget, con el objetivo de comprender el manejo de estados dinámicos en la interfaz.

La aplicación debe incluir al menos:

- Un botón interactivo (por ejemplo, ElevatedButton)
- Un widget visual (como un Container o Text) que cambie su apariencia al presionar el botón.

Al hacer clic en el botón, se debe ejecutar un cambio de estado utilizando setState(), provocando una modificación visible en pantalla, como por ejemplo:

- Cambio de color del contenedor.
- Cambio de tamaño (ancho o alto).
- Cambio de texto.
